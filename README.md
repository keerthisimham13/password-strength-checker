# 🔐 Password Strength Checker

A Python-based tool to check the strength of a password and provide suggestions for improvement.

## 🚀 Features
- Checks password length
- Detects uppercase and lowercase letters
- Verifies numbers and special characters
- Provides strength score (1–5)
- Gives suggestions to improve weak passwords

## 🛠️ Technologies Used
- Python
- Regular Expressions (re module)

## ▶️ How to Run

```bash
python password_checker.py
